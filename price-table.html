VPN Price Table (MMK)

Open this file in your browser and click "Download PNG" to save the price table image.

If the button doesn't work in your browser, open the page, right-click the table image and choose "Save image as..." or use the ImageMagick command shown below.

# ImageMagick (convert SVG → PNG)
magick convert -background none price-table.svg -resize 1400x560 price-table.png

# Or use Inkscape
inkscape price-table.svg --export-type=png --export-filename=price-table.png
